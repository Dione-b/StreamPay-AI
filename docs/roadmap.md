

# StreamPay AI Sprints Roadmap

## Sprint 1: Bootstrap & Setup

- Directory and module structure
- Initial setup of frontend (Next.js), backend (Node.js), smart contracts (Solidity)
- Development environment, CI/CD, Docker setup

## Sprint 2: MVP Core

- Implementation of StreamPayCore contract (streaming payments)
- Backend API: endpoints to create/query streams, Moralis/Chainlink integration
- Frontend: dashboard, wallet integration, stream creation flow
- Unit and integration tests

## Sprint 3: Compliance & Security

- KYC/AML integration (mock or real provider)
- LGPD consent flow
- Basic smart contract audit
- Monitoring, logging, alerts

## Sprint 4: Liquidity & Conversion

- Uniswap V3 integration (swaps, pools)
- Liquidity orchestration via backend
- Load and performance testing

## Sprint 5: UX & Observability

- UX improvements (onboarding, feedback, transaction status)
- Metrics dashboard, NPS, reports
- Final adjustments for limited production

## Sprint 6: Go Live & Pilot

- Deploy to testnet/mainnet (L2)
- Onboarding of real companies/freelancers
- Feedback collection, quick adjustments
- Preparation for external audit

## Acceptance Criteria

- Functional, secure, and tested MVP with minimum compliance
- Updated technical and product documentation
- Ready for limited production and future expansion
